Microsoft has unveiled Scout, its first Autopilot agent that works continuously in the background across Microsoft 365 applications, handling tasks like meeting coordination, calendar blocking, and risk identification without needing to be prompted each time. Built on OpenClaw open-source technology and operating under individual Entra identities, Scout respects organizational data protection policies while learning user work patterns through Work IQ capabilities. It represents a new category of always-on personal agents that understand how work gets done across apps and systems and take action proactively.
