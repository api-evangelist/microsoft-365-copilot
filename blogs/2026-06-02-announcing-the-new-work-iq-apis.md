Microsoft is launching Work IQ APIs for general availability on June 16, 2026, designed specifically for AI agents to interact with Microsoft 365 data and applications across four domains: Chat, Context, Tools, and Workspaces. The new APIs offer 2x faster processing speeds, 80% fewer tokens required than traditional APIs, and enterprise-scale support for high-frequency agent operations with built-in security and tenant trust boundary protection. Developers can begin building through public preview on GitHub, with consumption-based pricing measured in Copilot Credits and a new cost management dashboard in the Microsoft 365 admin center.
